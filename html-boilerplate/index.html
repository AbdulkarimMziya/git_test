HTML Basic Structure

We learned how to write html boilerplate for an HTML document.

The DOCTYPE

The DOCTYPE tells the browswer what version of HTML to render the the document. HTML syntax: < !DOCTYPE html >

HTML element

The root element of the document, meaning that every other element in the document will be a descendant of it.

Body element

Where we put important meta-information about our webpages, and stuff required for our webpages to render correctly in the browser

Body element

This is where all the content that will be displayed to users

Lists
